Известные системы идентификаторов

Следующие имена (URI) могут использоваться в качестве значения элемента system типа данных Identifier. Если URI определен здесь, он ДОЛЖЕН быть использован в предпочтении к любому другому механизму идентификации. Если система идентификаторов здесь не перечислена, корректный URI можно определить одним из следующих способов в указанном порядке:

  • в реестре HL7 OID
  • в документации, связанной с этим идентификатором
  • с помощью консультации с владельцем идентификатора
  • задать вопрос в почтовой рассылке HL7, касающейся словарей

Важное примечание: данный список URI является неполным и может быть изменен. Некоторые значения могут быть удалены, другие, вероятно, будут добавлены в ближайшие месяцы, так как HL7 учреждает формальные процессы вокруг URI в словарях.
