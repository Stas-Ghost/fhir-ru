Приветствуем на сайте FHIR®

FHIR - это стандарт обмена медицинскими данными, опубликованный HL7®. Примечание: сейчас идёт подготовка непрерывной интеграционной сборки к первому нормативному голосованию. Подробности на странице Голосование.

Впервые здесь?
Для ознакомления с FHIR посмотрите: Исполнительное резюме, Введение для программистов, Введение для медицинских работников или Введение для архитекторов ПО, Обзор стандарта FHIR и план развития, История версий. См. также открытую лицензию, оглавление, благодарности сообществу и поиск по спецификации.
