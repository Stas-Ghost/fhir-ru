Неразрешенные вопросы (Open Issues)

Для решения ряда вопросов особенно необходимо получить опыт реализации на этапе пробного использования стандартов:

  • Conformance (Соответствие): как используют системные профили
  • Order (Заказы): Необходимо определить дополнительные типы заказов
  • Resource Identification (Идентификация ресурсов): Управление идентификацией ресурсов в распределенных системах
  • Search (Поиск): Как следует стандартизировать текстовый поиск?
  • Search (Поиск): Насколько необходимо стандартизировать "_include"?
  • Security Labels (Метки уровня безопасности): Насколько политика защиты персональных данных может быть "заложена" в стандарте, какое влияние это окажет на саму спецификацию FHIR
  • Security (Безопасность): В каких клинических сценариях подразумевается использование цифровой подписи
  • Messaging (Обмен сообщениями): Обмен сообщениями еще не был проверен на практике в ходе Коннектатонов, поэтому приветствуется любой опыт практической реализации

В дополнение к базовому набору, в будущем планируется ввести следующие типы ресурсов:

  • Ресурс-уведомление (notification resource) для создания и фиксирования уведомлений в процессе взаимодействия сторон
  • Ресурс-предупреждение (alarm resource) для представления текущих проблем / вопросов, связанных с пациентом
  • Ресурс "человек" для ведения реестров населения
  • Ресурс "запланированная консультация / мед. процедура" (appointment resource) для планирования и управления запланированными в будущем медицинскими консультациями / процедурами.
  • Ресурсы, имеющие отношение к оплате (payment related resources) и в частности ресурс "Счет" (Account) для отслеживания платежей
  • Возможно, ресурсы для определения конкретных типов запросов (более точное определение, чем то, которое предлагает используемый сейчас профиль)
  • Один или несколько ресурсов для Advance Care Directive / Power of Attorney

Внимание: помимо указанных типов ресурсов могут вводиться и другие.

В дополнение к введению данных ресурсов планируется провести дополнительную работу по отслеживанию и формированию / отправке отчетов о клинических наблюдениях / жизненно-важных показателях здоовья и обеспечению возможности проведения удаленных консультаций / внедрения телемедицинских технологий.
