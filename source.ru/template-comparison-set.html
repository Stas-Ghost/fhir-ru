Сравнение <%pc.title%>

Это результат сравнения двух руководств по реализации <%pc.left%> и <%pc.right%>, сгенерированное, чтобы помочь сравнить взаимодействие реализаций, базирующихся на этих двух руководствах по реализации, или, в качестве альтернативы, чтобы помочь реализаторам понять, что требуется для соответствия обоим руководствам по реализации.

Сравнение состоит из серии сравнений между определениями ограничивающих условий, заданных в руководстве по реализации. Для каждого сравнения реализаторы могут просмотреть:

  • Серия сообщений от алгоритма сравнения. Ошибки показывают, что решения не смогут взаимодействовать с обоими руководствами по реализации (или что имеются структурные изъяны в определении по крайней мере одного из них)
  • Пересечение двух ограничивающих условий. Условия, которых должны придерживаться составители ресурса (клиент либо сервер), чтобы создавать содержимое, соответствующее обоим руководствам по реализации.
  • Объединение двух ограничивающих условий. Условия, которых должны придерживаться составители ресурса (клиент либо сервер), чтобы получать содержимое, соответствующее одному из руководств по реализации.
