См. полный перечень наборов значений, определенных в рамках FHIR.

Пояснение к столбцам, которые могут появиться на этой странице:

УровеньНекоторые перечни кодов, определенные FHIR, являются иерархическими: каждому коду назначен уровень. See Code System for further information.
ИсточникИсточник определения кода (когда набор значений построен из кодов, определенных где-то в другом месте).
КодКод (используемый в качестве кода в экземпляре ресурса). Если код выделен курсивом, значит его нельзя выбрать ('Abstract')
Значение для отображенияЗначение для вывода на экран (используемое в элементе display в Coding). Если значение для вывода на экран не указано, реализаторам следует показывать на экране не только код, а отобразить соответствующий концепт в свое приложение.
ОпределениеОбъяснение смыслового значения концепта.
КомментарииДополнительные указания к использованию кода.

Кроме того, эта страница будет включать в себя таблицы соответствия с кодами HL7 v2 или HL7 v3, где они были определены.
